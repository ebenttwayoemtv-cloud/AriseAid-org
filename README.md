AriseAid International - Global Charity Website Repository
Overview
This is the official GitHub repository for the AriseAid International public website. AriseAid is a humanitarian aid organization focused on sustainable development through health, education, and economic empowerment. The project is designed for maximum transparency and accountability and uses real-time data integration.

Innovative Features (World-Class Standard)
This website incorporates innovative features that align with leading global NGOs:

Real-Time Financial Transparency Tracker: An interactive simulator where donors can input any amount and immediately see the 85% Programs / 15% Overhead allocation breakdown, fostering trust and accountability.
Decentralized Data Model: Uses Firebase Firestore to display impact statistics and testimonials in real-time, proving data is always current and available.
WCAG Accessibility: Built with high color contrast, semantic HTML5, and ARIA attributes to ensure the site is fully accessible to all users, regardless of ability.
Enhanced Engagement Funnel: Features integrated newsletter sign-up and a clear, multi-tiered donation modal for focused giving.
Key Technical Features
Single-File Application: All HTML, CSS (Tailwind CSS), and JavaScript are contained within index.html for easy deployment.
Fully Responsive Design: Optimized for mobile, tablet, and desktop viewports.
Technology Stack:
Markup: HTML5 (Structure and content)
Styling: Tailwind CSS (Utility-first CSS framework)
Logic: JavaScript (Client-side functionality and real-time data handling)
Database: Firebase Firestore (Real-time data for stats and testimonials)
Icons: Lucide Icons (High-quality SVG icons)
Getting Started (Deployment)
Since this is a single, self-contained HTML file, it can be deployed instantly using services like GitHub Pages (recommended), Netlify, or Vercel.

Clone the Repository: Download the project files.
Deployment: Deploy the root directory containing the index.html file to your hosting service.
Custom Domain: For a professional look, set up a custom domain (e.g., www.ariseaid.org) in your chosen hosting service's settings.
Contribution Guidelines
AriseAid welcomes contributions from developers who share our mission.

Fork the Repository: Create your own copy of the repository.
Branch: Create a new branch for your feature (git checkout -b feature/new-tracker).
Single-File Mandate: Maintain the integrity of the project by keeping all HTML, CSS, and JS within the index.html file.
Testing: Test your changes using a local web server (e.g., Python's http.server) to ensure Firebase connectivity.
Pull Request: Submit a pull request to the main branch with a clear description of your improvements.
